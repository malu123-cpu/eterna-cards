# Eterna Cards

A wedding card design and management web application built using PHP and MySQL.
Developed as a BCA mini project.

## Features
- Admin, Customer, and Guest modules
- Browse wedding card designs
- Admin can add, edit, and delete cards
- Image upload and management
- User authentication

## Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## How It Works
The application uses PHP for backend logic and MySQL for storing card and user data.
Admins manage card designs through the admin panel, while users can browse cards
based on their access level.

## Setup
1. Install XAMPP
2. Place the project in `htdocs`
3. Configure database in `dboperation.php`
4. Start Apache and MySQL
5. Open `http://localhost/eterna-cards`
